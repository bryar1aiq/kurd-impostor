# kurd-impostor

This is an impostor game I created to play with my family.

**Play online:** [https://bryar1aiq.github.io/kurd-impostor/](https://bryar1aiq.github.io/kurd-impostor/)

## About

A Kurdish Sorani word game inspired by [wordimpostor.com](https://wordimpostor.com). Players receive a secret word—except the impostor(s), who must blend in and guess it. Optional "Mr White" role sees no word and must deduce it from discussion.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
